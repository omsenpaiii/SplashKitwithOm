TASK 1 OF WEEK 2:

This task was about creating a program by using variables, constants and basic knowledge of functions to collect user’s inputs and store the corresponding data in memory with the desired data types.
At that time, for me, the hard part to understand was the formation and declaration of a function. Another hard thing for me was to understand the use of constants and variables within the function.
So, the solution for this problem is quite easy, We just need to declare a global constant at the start of the program and we can use the same later on within our program.
For example, in this task, we’ve declared a global constant named STROUHAL with a value of 0.33 which we later used in the function calculate_air_speed as the following way: 

    double calculate_air_speed (int frequency, int amp) 
    {	
        return frequency*amp*0.01 / STROUHAL; 
    }
